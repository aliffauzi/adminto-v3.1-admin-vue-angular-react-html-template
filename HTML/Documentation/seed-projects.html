Seed Projects

We have built out the seed project for most of common languages and frameworks to make your job easy when you would like to use the Minton with any of these. We've covered Angular, React and Vue so far. As part of each of this seed, we have included structure, layouts, styles and inbuilt authentication, account related pages including registration, password, etc.

In order to get started with any of this seed, please refer to corresponding instructions

Angular Seed

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command npm install. This will install all required dependencies.
  3. Run ng serve for a dev server. Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions

React Seed

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command yarn install or npm install. This will install all required dependencies.
  3. Run yarn start or npm start for a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions

Vue Seed

  1. Go to command line and switch to your seed project directory
  2. Install the dependencies using command yarn install or npm install. This will install all required dependencies.
  3. Run yarn dev or npm dev for a dev server. Navigate to http://localhost:8000/. The app will automatically reload if you change any of the source files.
  4. Please refer to ReadMe file for more detailed instructions
